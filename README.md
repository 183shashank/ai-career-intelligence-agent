# AI Career Intelligence Agent 🚀

An AI-powered web application that matches your resume with job opportunities at any company.

## Features
- Upload Resume (PDF)
- Enter Company Name
- Optional Role & Location filter
- AI-powered job matching
- Top 5 ranked job recommendations

## Tech Stack
- Python
- Streamlit
- OpenAI API
- Tavily API

## How It Works
1. User uploads resume
2. App searches company job listings
3. AI analyzes resume & ranks jobs
4. Outputs top 5 matches

## Future Improvements
- Direct job scraping
- Real-time job description extraction
- Embedding-based matching
